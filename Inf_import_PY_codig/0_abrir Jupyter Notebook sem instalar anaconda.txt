abrir Jupyter Notebook sem instalar anaconda 

O Python Notebook (ou Jupyter Notebook) é uma aplicação web interativa que permite escrever e 
executar código Python em blocos, 
além de incluir textos, gráficos e tabelas — ideal para análise de dados,


1. Instalar o Jupyter via pip
Abra o terminal (Prompt de Comando, PowerShell ou terminal no VSCode) e digite:

pip install notebook

Esse comando apos ele abre a web => jupyter notebook

OU python -m notebook (esse func todas as vezes)
abre link
http://localhost:8888/tree


2 Rodar o Jupyter Notebook
Depois que a instalação terminar, execute:

jupyter notebook


3 Esse comando vai abrir o Jupyter Notebook automaticamente no seu navegador padrão, normalmente na URL:

http://localhost:8888/tree


Teste rápido
Você pode criar um arquivo .ipynb (notebook) novo direto na interface do navegador e começar a programar em Python.
